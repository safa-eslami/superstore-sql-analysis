# 📊 Superstore Sales Analysis (SQL Server)

## 📌 Project Overview

This project analyzes the Superstore dataset using SQL Server.

The goal is to answer real business questions through SQL queries and extract meaningful business insights from sales data.

This project covers essential SQL concepts commonly used in Data Analysis.

---

## 🛠️ Tools

- SQL Server
- SQL Server Management Studio (SSMS)

---

## 📚 SQL Concepts Used

- Aggregate Functions (SUM, COUNT)
- GROUP BY
- ORDER BY
- TOP
- Date Functions (YEAR, MONTH)
- Common Table Expression (CTE)
- Window Functions
- ROW_NUMBER()
- RANK()
